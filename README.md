Tutorial Java Game - Tank Wars
================

Team Members:

Nathan Shoemark
Milo Hawker
Ivelina Mincheva
Joyce Day-Hall
Jordan Hristov

Descption:

2-player projectile shooting game

Minimum Requirements:

Adjust angle and initial velocity of projectile
Movable tanks
Tank damage monitor

Additional Features:

Randomly generated terrain
Destroyable terrain
Zoomed view (scrollable)
Different Weapons
Multiple tanks per team
Tank 'power ups'
